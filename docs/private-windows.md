# Enable Private Windows Access

If the popup shows the error "Must Enable Private Windows - help", the extension does not have permission to run in Private Windows. Enable it once and the error will go away.

## Steps

1. Open the Add-ons Manager: type `about:addons` in the Firefox address bar.
2. Go to **Extensions** and find **Fiversox**.
3. Click the extension to open its details page.
4. Under **Run in Private Windows**, select **Allow**.
5. Reopen the extension popup.
